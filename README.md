Screen locking script with blurring.

### Installation
Depends on:
- bash
- i3lock
- ImageMagick

Copy the `blurlock` script to `/usr/bin/blurlock`.

### Usage
```
blurlock [RADIUS]

    RADIUS [optional]: the blur radius, defaults to 2. Higher values mean stronger blur.
```
